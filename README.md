# 釜山Boy Translator

## 概要・説明
**釜山Boy Translator** は、入力された日本語を「親しみやすく、男らしい釜山の20代男性風の韓国語（慶尚道方言）」に意訳するユニークなAI翻訳Webアプリケーションです。

釜山方言への意訳だけでなく、標準的な日韓・韓日翻訳、言語を自動判定する多言語翻訳、そしてAIが最適な言語を選択して翻訳するスマートモードなど、実用的な翻訳機能も備えています。

本アプリは単一のHTMLファイルで構成された完全なフロントエンド・アプリケーションです。Googleの最新AIモデル「Gemini API」をブラウザから直接呼び出し、サーバーレスかつ高速なストリーミング翻訳を実現しています。また、APIキーや翻訳履歴などのデータはすべてお使いの端末（IndexedDB）に安全にローカル保存されます。

## 主な機能一覧
- **5つの高度なAI翻訳モード**
  - `釜山意訳` : 釜山の若者らしい情熱的で親しみやすい方言に意訳＋解説
  - `日 ➔ 韓` : 日本語から自然な標準韓国語へ翻訳
  - `韓 ➔ 日` : 韓国語から自然な日本語へ翻訳
  - `自動翻訳` : 入力言語を自動判定し、指定した言語（英語、中国語、スペイン語など）へ翻訳
  - `スマート` : AIが文脈を読み取り、最も適切な言語へ自動翻訳
- **プロンプトのカスタマイズ**
  - 各モードにおけるAIへの指示（システムプロンプト）をユーザー自身で自由に変更可能。
- **高速ストリーミング生成**
  - Web WorkerとSSE (Server-Sent Events) を活用し、UIをブロックすることなくAIの回答をリアルタイムでストリーミング表示します。
- **クリップボード連携・自動翻訳**
  - テキストをペーストした瞬間に自動でAI生成を開始するスマートなショートカット機能を搭載。
- **強力なローカルデータ管理**
  - **履歴機能**: 過去の翻訳結果を自動保存し、いつでも呼び出し可能。
  - **自動バックアップ**: 生成完了時に直近5世代のバックアップを自動作成。
  - **インポート/エクスポート**: 設定や履歴データをJSONファイルとして入出力し、端末間でのデータ移行が可能。
- **PWA (Progressive Web App) 対応**
  - スマートフォンのホーム画面に追加することで、ネイティブアプリのようなフルスクリーン体験が可能です。

## 使用技術・ライブラリ
本プロジェクトは、モダンなWeb標準技術と軽量なCDNライブラリを組み合わせて構築されています。

- **コア技術**: HTML5 / CSS3 / Vanilla JavaScript
- **スタイリング**: [Tailwind CSS](https://tailwindcss.com/) (CDN)
- **アイコン**: [FontAwesome](https://fontawesome.com/) (CDN)
- **マークダウンレンダリング**: [Marked.js](https://marked.js.org/) & [DOMPurify](https://github.com/cure53/DOMPurify) (HTMLサニタイズ)
- **AI / API**: [Google Gemini API](https://ai.google.dev/) (SSE Streaming API)
- **データ永続化**: IndexedDB API (非同期ラッパークラスを独自実装)
- **バックグラウンド処理**: Web Workers API
- **オフライン・PWA対応**: Service Worker & Web App Manifest (Data URI形式で動的生成)

## セットアップ・使い方

### 1. 準備 (APIキーの取得)
本アプリを利用するには、Googleの無料AI APIキーが必要です。
1. [Google AI Studio](https://aistudio.google.com/) にアクセスし、Googleアカウントでログインします。
2. 「Get API key」または「Create API key」から、新しいAPIキーを作成してコピーします。

### 2. アプリの起動
本アプリはサーバー構築が不要です。
提供されたHTMLファイルをローカルのブラウザ（Chrome, Safari, Edgeなど）で直接開くか、GitHub PagesやVercelなどの静的ホスティングサービスにデプロイしてアクセスしてください。

### 3. 初期設定
1. アプリ画面右下の「**歯車アイコン**（設定・データ管理）」をクリックします。
2. 「Gemini API Key」の入力欄に、手順1で取得したAPIキーを貼り付けます。
   *(※ APIキーは端末のブラウザ内部にのみ保存され、外部サーバーへは送信されません)*
3. 必要に応じて「AIモデル」を選択します（デフォルトの `Gemini 2.5 Flash` が超高速で推奨です）。
4. 設定画面の右上「×」ボタンで閉じます（自動的に保存されます）。

### 4. 使い方
- **翻訳の実行**
  中央の入力エリアに翻訳したいテキストを入力し、画面下部の「**🪄 変換する**」ボタンを押します。
- **ワンタップ翻訳**
  「📋 ペーストして翻訳」ボタンを押すと、クリップボードの内容を貼り付けると同時に翻訳が開始されます。
- **モードの切り替え**
  入力エリア上部のタブ（釜山意訳、日➔韓、韓➔日、自動翻訳、スマート）をクリックしてモードを切り替えます。
- **履歴の確認**
  画面左下の「**時計アイコン**」をクリックすると、過去の翻訳履歴の一覧が表示され、タップして復元することができます。
