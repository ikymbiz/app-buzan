# 釜山Boy Translator

## 概要・説明
釜山Boy Translatorは、Google Gemini APIを活用したブラウザ完結型の多機能AI翻訳Webアプリケーションです。
最大の特徴は、日本語のテキストを「釜山の20代男性風（男らしく親しみやすい方言）」の韓国語に意訳するユニークなモードを備えている点です。方言への変換に加えて、標準的な「日韓」「韓日」翻訳や、複数言語に対応した「自動翻訳モード」も搭載しています。

AIの出力をリアルタイムで表示するストリーミング機能、システムプロンプトのカスタマイズ機能、そしてAPIキーや履歴データをブラウザ内（IndexedDB）に留めるセキュアなローカルファースト設計を採用しており、実用性とエンターテインメント性を兼ね備えた高品質な翻訳ツールです。

## 主な機能一覧
- **4つの翻訳・意訳モード**
  - **釜山意訳**: 入力された日本語を釜山方言（慶尚道方言）に変換し、「方言での意訳」「標準的な直訳」「解説や一言」をMarkdown形式で出力。
  - **日 ➔ 韓 / 韓 ➔ 日**: 自然で高精度な標準語翻訳。
  - **自動翻訳**: 言語を自動判定し、指定したターゲット言語（英語、韓国語、日本語、中国語、スペイン語、フランス語）へ翻訳または文章の校正を実行。
- **Gemini API連携と高速レスポンス**
  - Gemini 2.5 Flash / 1.5 Pro / 1.5 Flashモデルから選択可能。
  - Web WorkerとSSE（Server-Sent Events）を活用した、UIをブロックしないリアルタイムなストリーミングテキスト生成。
- **柔軟なカスタマイズと自動化**
  - AIの振る舞いを決定する「システムプロンプト」をモードごとに自由に編集・リセット可能。
  - テキストをペーストした直後に自動で翻訳を開始する機能。
- **セキュアなローカルデータ管理**
  - APIキー、設定状態、翻訳履歴はすべてブラウザのIndexedDBに保存され、外部サーバーには送信されません。
  - 翻訳実行ごとの履歴保存機能と、直近5世代の自動バックアップ機能。
  - JSONファイルを用いたデータのエクスポート・インポート機能。
- **PWA（Progressive Web App）対応**
  - スマートフォンやPCのホーム画面に追加して、ネイティブアプリのように利用可能。

## 使用技術・ライブラリ
- **フロントエンド言語**: HTML5, Vanilla JavaScript, CSS3
- **スタイリング**: Tailwind CSS (CDN)
- **アイコン**: FontAwesome
- **Markdown解析・サニタイズ**: Marked.js, DOMPurify
- **データベース**: IndexedDB (ラッパークラスによる非同期処理)
- **並行処理**: Web Worker API (AIリクエスト処理用)
- **外部API**: Google Gemini API
- **その他**: Service Worker (PWA実装)

## セットアップ・使い方

### 1. 準備（APIキーの取得）
本アプリを利用するには、GoogleのGemini APIキーが必要です。
[Google AI Studio](https://aistudio.google.com/) にアクセスし、Googleアカウントでログインして無料のAPIキーを取得してください。

### 2. アプリの起動
特別なビルド環境（Node.jsなど）は不要です。
提供されたHTMLコードを `index.html` として保存し、以下のいずれかの方法で開いてください。
- ブラウザに直接ドラッグ＆ドロップして開く。
- VS Codeの Live Server 拡張機能などでローカルホストを立ち上げて開く。
- GitHub Pages, Vercel, Cloudflare Pages などの静的ホスティングサービスにデプロイする。

### 3. 初期設定
1. アプリ画面の右下にある **「設定アイコン（歯車）」** をクリックします。
2. 取得した **Gemini API Key** を入力欄にペーストします。（※入力したキーは端末内にのみ保存されます）
3. 利用したいAIモデル（推奨: Gemini 2.5 Flash）を選択します。
4. 設定モーダルの右上にある「×」ボタンを押して画面を閉じます。

### 4. 基本的な使い方
1. 画面中央のモードセレクターから、希望の翻訳モード（釜山意訳、日 ➔ 韓、韓 ➔ 日、自動翻訳）を選択します。
2. 「Input」エリアに翻訳したいテキストを入力、または貼り付けます。
3. 画面下部中央の **「変換する（魔法の杖アイコン）」** ボタンをクリックします。
4. 「Output」エリアにAIからの回答がリアルタイムでレンダリングされます。
5. 翻訳結果の右上にあるコピーボタンから、テキストをクリップボードにコピーできます。

**その他の便利機能:**
- **履歴の確認**: 左下の「時計アイコン」をクリックすると、過去の翻訳履歴を閲覧し、再読み込みさせることができます。
- **プロンプト変更**: 設定画面から、各モードのAIへの指示（プロンプト）を自分好みに書き換えることが可能です。デフォルトに戻すこともボタン一つで簡単に行えます。
