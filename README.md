# 釜山Boy Translator

## 概要・説明
「釜山Boy Translator」は、Google Gemini APIを活用したブラウザ完結型の翻訳Webアプリケーションです。
最大の特徴は、ユーザーが入力した日本語を**韓国・釜山出身の20代男性風（男らしく親しみやすい方言）**に意訳する「釜山意訳モード」です。単なる翻訳にとどまらず、方言特有のニュアンスや解説もあわせて出力されるため、楽しく韓国語（慶尚道方言）に触れることができます。

通常の「日 ➔ 韓」「韓 ➔ 日」の標準翻訳機能も備えており、モバイルファーストなUIデザインとPWA対応により、スマートフォンのネイティブアプリのような感覚で快適に利用できます。

## 主な機能一覧
- **3つの翻訳モード**
  - **釜山意訳**: 日本語を釜山方言（慶尚道方言）に意訳し、直訳・解説を交えて出力します。
  - **日 ➔ 韓**: 日本語から自然な標準韓国語への翻訳。
  - **韓 ➔ 日**: 韓国語から自然な日本語への翻訳。
- **リアルタイムな生成表示 (ストリーミング)**
  - Web WorkerとSSE（Server-Sent Events）を利用し、AIの生成結果を1文字ずつリアルタイムで画面に表示します。
- **カスタマイズ可能なプロンプト**
  - 設定画面から各モードのAIシステムプロンプト（指示内容）を自由に編集でき、好みのキャラクターや出力形式に調整可能です。
- **完全ローカル保存のプライバシー配慮**
  - APIキーや翻訳履歴などのデータは、ブラウザのIndexedDB内にのみ保存され、外部サーバーへは送信されません。
- **履歴・バックアップとデータ移行**
  - 過去の翻訳履歴の閲覧・復元機能。
  - 直近5世代の自動バックアップ機能。
  - JSONファイルによるデータのインポート / エクスポート機能。
- **PWA（Progressive Web App）対応**
  - スマートフォンの「ホーム画面に追加」機能を利用することで、全画面のアプリとして動作します。

## 使用技術・ライブラリ
- **フロントエンド**: HTML5, JavaScript (Vanilla JS)
- **スタイリング**: Tailwind CSS (CDN経由)
- **アイコン**: FontAwesome
- **テキスト処理**: 
  - Marked.js (Markdownのパース)
  - DOMPurify (HTMLのサニタイズ)
- **データストレージ**: IndexedDB (独自の非同期ラッパークラスにて実装)
- **AI・API**: Google Gemini API (`gemini-2.5-flash` モデル)
- **その他**: Web Workers (バックグラウンドでのAPI通信), Service Worker (PWA対応), Wake Lock API (生成中のスリープ防止)

## セットアップ・使い方

### 1. アプリの起動
本プロジェクトはバックエンドサーバーを必要としません。
提供されているHTMLファイルをダウンロードし、お使いのWebブラウザ（Chrome, Safari, Edgeなど）で直接開くか、GitHub Pagesなどの静的ホスティングサービスにデプロイするだけで動作します。

### 2. 初期設定（APIキーの登録）
AI機能を利用するために、Google GeminiのAPIキーが必要です。
1. [Google AI Studio](https://aistudio.google.com/) にアクセスし、APIキーを取得します。
2. アプリ画面右下の「歯車アイコン（設定）」をタップして設定モーダルを開きます。
3. 「Gemini API Key」の入力欄に取得したキーを入力します。（※キーはブラウザ内部にのみ保存されます）
4. モーダル右上の「×」ボタンで設定画面を閉じます。

### 3. 基本的な使い方
1. **モード選択**: 画面上部のタブから「釜山意訳」「日 ➔ 韓」「韓 ➔ 日」のいずれかを選択します。
2. **テキスト入力**: 「Input」エリアに翻訳・意訳したいテキストを入力します。
3. **変換の実行**: 画面下部中央の「魔法の杖アイコン（変換する）」ボタンをタップします。
4. **結果の確認**: AIが生成したテキストが「Output」エリアにリアルタイムで表示されます。出力エリア右上のアイコンからテキストをクリップボードにコピーできます。

### 4. 履歴とバックアップの利用
- **履歴の確認**: 画面左下の「時計アイコン」から、過去の翻訳履歴を確認し、タップして入力状態を復元できます。
- **バックアップ・エクスポート**: 設定画面下部から、現在のデータ（履歴や設定）をJSONファイルとしてエクスポートしたり、機種変更時にインポートしてデータを引き継ぐことが可能です。
