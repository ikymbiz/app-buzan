# 釜山Boy Translator

## 概要・説明
「釜山Boy Translator」は、日本語のテキストを韓国・釜山の20代男性風（親しみやすく、男らしく、少し強気で情熱的な慶尚道方言）に意訳してくれるAI翻訳Webアプリケーションです。
単なる翻訳にとどまらず、釜山方言への意訳、標準的な直訳、そして釜山男らしい解説までをセットにして提供します。また、通常の「日➔韓」「韓➔日」翻訳モードも備えています。

Google Gemini APIを利用した高速なストリーミング生成、端末内にデータを安全に保存するローカルファーストな設計、そしてスマートフォンでネイティブアプリのように使えるPWA（Progressive Web App）に対応したモダンで直感的なUIが特徴です。

## 主な機能一覧
- **3つの翻訳・意訳モード**
  - **釜山意訳**: 釜山方言での意訳、直訳、釜山男らしい一言解説の3段構成でリッチに出力。
  - **日 ➔ 韓**: 日本語から自然な標準韓国語への翻訳。
  - **韓 ➔ 日**: 韓国語から自然な日本語への翻訳。
- **AIリアルタイム生成（ストリーミング）**
  - Web WorkerとSSE（Server-Sent Events）を活用し、AIの回答を待たずに文字が次々と表示されます。生成の途中停止も可能です。
- **履歴と自動保存（ローカルDB）**
  - 入力テキストや翻訳結果はブラウザのローカルDB（IndexedDB）に自動保存されるため、誤って画面を閉じても安心です。
- **高度なバックアップ・データ管理**
  - 翻訳成功時に直近5世代のバックアップを自動作成。
  - JSON形式でのエクスポート・インポート機能により、端末間でのデータ移行も簡単です。
- **プライバシー＆セキュリティ配慮**
  - APIキーは端末内部（IndexedDB）にのみ保存され、開発者や外部のサーバーへは一切送信されません。
- **PWA・モバイル最適化**
  - スマホのホーム画面に追加可能。画面のSafe Areaに対応し、Glassmorphismを取り入れた美しいUIデザインを採用しています。
  - 翻訳中は画面がスリープしないよう Wake Lock API で制御されます。

## 使用技術・ライブラリ
- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript
- **スタイリング**: Tailwind CSS (CDN), FontAwesome (Icon)
- **AI・API**: Google Gemini API (`gemini-2.5-flash:streamGenerateContent`)
- **テキスト処理**: Marked.js (Markdownパース), DOMPurify (HTMLサニタイズ)
- **ブラウザAPI・その他**: 
  - IndexedDB (ローカルデータ永続化)
  - Web Workers (メインスレッドをブロックしない非同期API通信)
  - Service Worker & PWA Manifest (インストール・オフラインキャッシュ)
  - Wake Lock API (画面スリープ防止機能)

## セットアップ・使い方

### 準備
本アプリはブラウザ上で完結するクライアントサイドWebアプリですが、AIによる翻訳機能を利用するために **Google Gemini APIキー** が必要です。
1. [Google AI Studio](https://aistudio.google.com/) にアクセスし、ご自身のGoogleアカウントでログインしてAPIキーを取得してください。

### 起動方法
1. 提供されたHTMLコードを `index.html` という名前で保存します。
2. VS Codeの「Live Server」拡張機能など、ローカルWebサーバーを使用してファイルを開いてください。
   *※Web WorkerやPWA（Service Worker）の機能制限を回避するため、`file://` プロトコルではなく `http://localhost...` または `https://...` 経由での実行を推奨します。*

### 使い方
1. **APIキーの設定**
   - アプリ起動後、画面右下の「歯車アイコン（設定）」をタップします。
   - 取得した Gemini API Key を入力欄に貼り付けます（自動的に保存されます）。
   - 右上の「×」ボタンで設定画面を閉じます。
2. **モードの選択**
   - 画面上部のタブから「釜山意訳」「日 ➔ 韓」「韓 ➔ 日」のいずれかを選択します。
3. **テキストの入力と翻訳**
   - 「Input」エリアに翻訳したいテキストを入力します。
   - 画面下部中央の「✨ 変換する」ボタンをタップすると、AIがリアルタイムで結果を生成します。
4. **結果のコピー・履歴の確認**
   - 出力されたテキストは、「Output」エリア右上のコピーアイコンをタップすることでクリップボードにコピーできます。
   - 画面左下の「時計アイコン（履歴）」をタップすると、過去の翻訳履歴を一覧で確認でき、選択すると入力・出力状態を復元できます。
5. **バックアップと復元**
   - 設定画面（歯車アイコン）から、手動でのデータエクスポート（JSON保存）やインポート、自動生成されたバックアップからの復元が行えます。定期的にエクスポートしておくことで、ブラウザのキャッシュクリア時も安心です。
